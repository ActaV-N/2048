# Usage

**If you want to use virtualenv**

```sh
python -m venv .
source ./bin/activate
```

**Install**

```sh
pip install -r requirements.txt
```

**Run server**

```sh
fastapi dev main.py
```

**2048**
Just open `gui.html` in the root directory.\
If you want to reset game state, clean up the local storage of browser.
